# Rock-Paper-Scissors

- An AI based Rock Paper Scissors game where you can play real time with an AI using your webcam

- Requires a Webcam to detect player's hand gestures 

- Developed using 
  >- Python
  >- OpenCV
  >- CVzone
  >- Mediapipe
